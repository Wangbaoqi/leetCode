# Nate Algorithms

每日一题，怒刷算法

本仓库主要使用`TS` + `Jest`来编写算法题目。

使用下面脚本来进行单元测试

```bash
$ npm run test  // or yarn test
```

可以随时添加测试用例，仓库中已有的的算法题都已经通过了**LeetCode**的测试用例。
