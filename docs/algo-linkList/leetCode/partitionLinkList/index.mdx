:::info
给你一个链表的头节点 head 和一个特定值 x ，请你对链表进行分隔，使得所有 小于 x 的节点都出现在 大于或等于 x 的节点之前。

你应当 保留 两个分区中每个节点的初始相对位置。

```mermaid
  direction LR

  style d1 fill:#f9f,color:#fff
  style f1 fill:#f9f,color:#fff
  style e1 fill:green,color:#fff
  end

  before ---> after
  
```
