<Note>

将两个升序链表合并为一个新的 升序 链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。

```md
flowchart LR;

  subgraph node1
  direction LR
  A(1) --> B(2) --> C(4)
  end

  subgraph node2
  direction LR
  A1(1) --> B1(3) --> C1(4)
  end

  subgraph node3
  direction LR
  A2(1) --> A3(1) --> B2(2) --> C2(3) --> C3(4)  --> C4(4)
  end

  node1-- merge ---node2 ---> node3

```

</Note>
