<Note>

给你一个链表的头节点 head 。

对于列表中的每个节点 node ，如果其右侧存在一个具有 严格更大 值的节点，则移除 node 。

返回修改后链表的头节点 head 。

</Note>

示例：

```
输入：head = [5,2,13,3,8]
输出：[13,8]
解释：需要移除的节点是 5 ，2 和 3 。
- 节点 13 在节点 5 右侧。
- 节点 13 在节点 2 右侧。
- 节点 8 在节点 3 右侧。
```

## 单调栈

利用单调栈可以找出大于当前节点的下一个节点，类似于[链表中的下一个更大节点](/docs/algo-linkList/leetCode/nextLargeNodes/)

这里不同的是，需要删除当前节点，那么就需要存储当前节点的前一个节点。

这里有个技巧，就是利用**哨兵节点**, 并且将哨兵节点初始化压入单调栈中，如果找到大于当前的节点的下一个节点，
此时单调栈中之剩下哨兵节点了，则可以进行删除节点了。

时间复杂度**O(n)**，空间复杂度**O(n)**

## 反转链表遍历

利用反转链表，首先将`head`进行反转，此时当前节点的下一个更大节点在当前节点的前面，则可以直接遍历，如果
当前节点大于下一个节点，则删除下一个节点。

最后将`head`进行反转就是最后的删除后的链表。

时间复杂度**O(n)**，空间复杂度**O(1)**
